Topdown adventure game with Löve
--------------------------------

To get Löve, see the instructions for your platform on [its
homepage](https://love2d.org/).

To try out the game, run (requires Löve 11.2):
```bash
$ love game
```

Similarly, to run the editor:
```bash
$ love editor
```
